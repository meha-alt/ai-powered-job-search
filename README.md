# ai-powered-job-search


### Overview

An AI-driven system that analyzes resumes to extract skills, aligns them with relevant job opportunities using semantic matching, and generates personalized outreach messages and resume improvement suggestions.

### Problem

Traditional job platforms rely on keyword matching, making it difficult for users to find relevant roles and tailor outreach based on their actual skills.

### Solution

Developed a system that extracts user skills from resumes, matches candidates with relevant job opportunities, generates personalized outreach messages, and suggests resume improvements.

### Approach

* Built a resume parsing pipeline to extract skills and key information
* Implemented semantic job matching using embeddings
* Integrated Adzuna API for real-time job listings
* Designed an LLM-based system to generate outreach messages and resume improvement suggestions
* Structured outputs for clarity and actionability

### Impact

* Improved job-role relevance beyond keyword-based matching
* Reduced manual effort in job search and outreach
* Enabled a more personalized and efficient user experience

### Tech Stack

Python, NLP (embeddings), Adzuna API, LLMs, Streamlit (if applicable)
